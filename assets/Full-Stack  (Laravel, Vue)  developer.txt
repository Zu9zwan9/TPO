OpenGeeksLab – аутсорсингова компанія, яка займається web та mobile розробкою проектів з нуля.

Працюємо у сферах FinTech, Healthcare, Social Media, Entertainment, Fitness, Logistics, Hospitality. Наш основний стек технологій: Node.js, React, React Native, Flutter.

Запрошуємо до нашої команди приєднатися досвідченого Full-Stack розробника.

Необхідні навички:
- Досвід комерційної розробки від 2-3 років з PHP;
- Досвід роботи з Laravel PHP від 2 років;
- Досвід роботи з Vue.JS від 2 років;
- Досвід роботи з базами даних PostgreSQL, MySQL, MongoDB;
- Досвід з Redis, Elasticsearch;
- Розуміння OOP та принципів Solid;
- Досвід роботи з API (REST API через JSON);
- Git;
- Знання англійської мови не нижче рівня Upper-Intermediate. Буде плюсом досвід спілкування із клієнтом.
